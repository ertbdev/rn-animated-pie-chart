# rn-animated-pie-chart

Animated Pie Chart for React Native

## Installation

```sh
npm install rn-animated-pie-chart
```

## Usage

```js
import { multiply } from 'rn-animated-pie-chart';

// ...

const result = await multiply(3, 7);
```
