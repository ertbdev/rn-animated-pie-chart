# rn-animated-pie-chart

Animated Pie Chart for React Native

## Installation

```sh
npm install rn-animated-pie-chart
```

## Usage

```js
import { PieChart, AnimatedPieChart } from 'rn-animated-pie-chart';

// ...

    <PieChart data={[10, 20]} />
    <AnimatedPieChart data={[10, 20, 30]} />
```
